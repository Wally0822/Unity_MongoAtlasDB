# 개요 :
Unity 게임 엔진과 MongoDB Atlas 클라우드 데이터베이스 간의 효율적인 연동을 목표로 진행되었습니다. MongoDB Atlas와의 원활한 통신을 위해 Unity에 MongoDB Atlas DLL 플러그인을 사용하였습니다. 게임 내에서 사용자의 랭킹 정보를 실시간으로 저장하고, 해당 정보를 기반으로 게임 내 랭킹을 확인 및 적용할 수 있게 되었습니다.
